Every collection has a first page, and the first page is always a little
dishonest. It pretends the work started here, on this date, with this
sentence — when in truth the writing began long before, in fragments and
false starts that never earned a filename.

This is a **sample entry**. Replace it with your own. The system reads any
Markdown file you drop into `writings/essays/` and places it in the
collection by date, so the order takes care of itself.

> The order of a life is only visible in retrospect; chronology is the
> kindest fiction we tell about it.

A few things this format supports, so you can write naturally:

- *Italic* and **bold** for emphasis.
- Section headings within a piece.
- Block quotes, like the line above.

## A heading inside the essay

Paragraphs are justified and set in a serif face, the way a printed book
would set them. Write as long or as short as you like — each file is one
piece, and each piece finds its own place in time.
